## 🧪 Como Rodar o Projeto Localmente

### Pré-requisitos
* Node.js (v18+)
* npm ou yarn

### Passos:

```bash
# 1. Clone o repositório
git clone https://github.com/ruitobias-eng/dibitechsite.git

# 2. Acesse o diretório
cd dibitechsite

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse a aplicação no seu navegador em `http://localhost:3000`.

## 📦 Como Publicar no GitHub Pages

Para realizar o deploy estático no **GitHub Pages**:

1. Garanta que o arquivo `vite.config.ts` possua a propriedade `base: './'`.
2. Gere a build estática de produção:
   ```bash
   npm run build
   ```
3. O conteúdo da pasta `dist/` gerada pode ser publicado na branch `gh-pages` manualmente ou via **GitHub Actions**.
